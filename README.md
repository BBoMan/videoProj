# 편집해조(15조) 캡스톤 디자인 프로젝트 : Shorts Editing

## 프로젝트 정의
현재 많은 인터넷 방송인들은 방송 중 발생한 재미있는 상황을 긴 영상과 짧은 영상(쇼츠) 형태로 편집하여 유튜브에 업로드하고 있습니다. 그러나 대부분의 초보 인터넷 방송인들은 수익이 크지 않아 전문 편집자를 고용하기 어렵습니다. 그래서 직접 편집을 수행하고 있는 초보 인터넷 방송인이 대부분입니다. 이러한 이유로 콘텐츠 제작에 충분히 집중하지 못하고, 영상 편집에 상당한 시간을 소모하는 어려움을 겪고 있습니다. 뿐만 아니라 긴 영상을 쇼츠로 제작할 때, 전체 영상을 일일이 확인하며 편집 포인트를 찾는 것은 매우 번거롭습니다. 특히 취미나 부업으로 인터넷 방송을 하는 경우, 편집에 할애할 시간이 부족하여 아예 영상을 업로드하지 못하는 상황도 발생합니다. 그렇기에 저희는 이 프로젝트를 이용해 영상 편집 과정을 자동화하고 효율화하고자 합니다.

## 프로젝트 개발 환경
- C++
- C#
- Python

## 기대효과
- 일차적 사용자와 잠재적 사용자가 제공받을 수 있는 기능
   일차적 사용자인 초보 인터넷 방송인은 유튜브, 인스타그램, 틱톡과 같은 플랫폼에 자신의 영상을 큰 노력을 들이지 않고 업로드할 수 있게 됩니다. 잠재적 사용자인 대형 인터넷 방송인의 편집자는 편집을 할 때 주목해야 하는 부분을 쉽게 알 수 있어 생산성을 올릴 수 있을 것입니다.

- 문제 해결 여부를 판단할 수 있는 기준
   생성된 영상 편집물의 퀄리티와 조회수

- 프로젝트 결과물이 갖는 의미
영상 편집에 벽을 느낀 인터넷 방송인들에게 서비스를 제공해 다양한 곳에 자신을 알릴 수 있는 기회를 제공할 수 있으며, 생성된 영상들로 사람들에게 재미와 행복감을 주어 사회 전반적으로 삶의 만족도가 올라갈 것을 기대합니다.

- 프로젝트를 통해 성취하고자 하는 연관된 목표
  프로젝트를 더욱 발전시키기 위해 수익 창출을 목표하고 있습니다.
